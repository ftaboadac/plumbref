# Current Claim

Plumbref verifies AI coding-agent claims against local repository evidence before
users rely on them.

The narrow, currently preferred public claim is:

> Plumbref gives you a careful-agent workflow even when you cannot trust that
> the agent will naturally behave like a careful senior engineer.

Claims to avoid unless a specific demo proves them:

- Plumbref always beats careful agents.
- Plumbref always saves tokens.
- Plumbref proves code behavior automatically.
- Plumbref replaces human review.

Important nuance:

- The coding agent still extracts claims and reasons over evidence.
- Plumbref supplies local tools, templates, search requirements, source snippets,
  conservative status semantics, broad-claim detection, redaction, budgets, and
  inspectable report artifacts.
